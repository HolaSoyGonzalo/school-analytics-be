### Configuration

In order to correctly deploy the application, you need a `.env` file.

In details:

* *app_port*: port exposed by app from which endpoints can be reached;
* *db_host*: database host
* *db_user*: database user
* *db_name*: database name
* *db_port* database port
* *db_dialect*: database type
* *db_password*: database password for configured user
* *db_ssl_connect*: determines if SSL connections are required in order to communicate with DB
* *jwt_key*: secret jwt key
* *jwt_refresh_key*: secret jwt refresh key
